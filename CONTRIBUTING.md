# Contributing to Pokémon Verdant

Thank you for your interest in Pokémon Verdant!

Pokémon Verdant is currently a personal hobby and learning project. I’m using this fork to practice ROM hacking, development workflow, testing, project management, and release planning.

Because of that, I’m not accepting outside contributions, pull requests, or issue assignments for this repository at this time.

## Issues

GitHub issues in this repository are being used mostly as a personal planning and testing tracker.

They may include:

- Feature ideas
- Testing notes
- Balance plans
- Bug notes from my own playthrough
- Future roadmap items
- Documentation and companion-site planning

Please do not claim issues or begin work on them unless I have specifically invited collaboration.

## Pull Requests

Pull requests are not currently being accepted for Pokémon Verdant.

If you are interested in contributing to the engine this project is based on, please consider contributing to the parent project instead:

- RHH `pokeemerald-expansion`: https://github.com/rh-hideout/pokeemerald-expansion

That project has its own contribution process, issue tracker, pull request guidelines, documentation, and community channels.

## Suggestions

I appreciate enthusiasm and kindness around the project. However, because this is currently a personal learning project, I may not be able to respond to suggestions or requests.

If I open the project to outside contributions in the future, I’ll update this file with new guidelines.

Thanks for understanding!
